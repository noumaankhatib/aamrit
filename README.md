# Aamrit — Premium Alphonso Mangoes

Website for **Aamrit**, a premium Alphonso mango brand by Yeskay Mango Farms Pvt. Ltd. — family-grown across 10 farms in Ratnagiri & Raigad since 2008.

## Run locally

```bash
npm install
npm run dev
```

Open <http://localhost:3000>.

## Build for production

```bash
npm run build
npm run start
```

## Tech stack

- **Next.js 15** (App Router, RSC)
- **TypeScript**
- **Tailwind CSS** (custom theme: gold / saffron / cream / leaf / charcoal)
- **Framer Motion** — UI animations
- **GSAP + ScrollTrigger** — scroll storytelling
- **Lenis** — smooth scrolling on desktop
- **next/font** — optimized Playfair Display + Poppins
- **next/image** — AVIF/WebP image optimization
